# 祈福点灯图片说明

此目录用于存放祈福点灯相关的图片文件。

## 必需文件

### 页面标头
- `祈福点灯.png` - 页面顶部标头图片（推荐尺寸：1920x400px）

### 灯笼图片
用户上传的汉字命名灯笼图片，例如：
- `平安灯.png`
- `财运灯.png`
- `健康灯.png`
- `爱情灯.png`
- `事业灯.png`
- `学业灯.png`
- `家庭灯.png`
- `智慧灯.png`
- `福气灯.png`
- `长寿灯.png`

## 图片要求

1. **格式**：PNG格式，支持透明背景
2. **尺寸**：推荐正方形或接近正方形比例（如 800x800px）
3. **质量**：清晰度高，适合网页展示
4. **命名**：使用中文名称，以`.png`结尾

## 自动检测

系统会自动扫描此目录中的所有汉字命名PNG文件，并生成对应的祈福灯卡片。文件名将作为灯的名称显示在页面上。

## 现有文件

- `p1.jpg` - 福运灯（备用）
- `p2.jpg` - 安康灯（备用）
- `p3.jpg` - 财源灯（备用）
- `p4.jpg` - 事业灯（备用）

当没有找到汉字命名的PNG文件时，系统将使用这些备用图片。
