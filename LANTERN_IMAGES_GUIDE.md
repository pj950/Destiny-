## 需要上传的图片

### 1. 页面标头图片
- **文件名**: `祈福点灯.png`
- **位置**: `/public/images/祈福点灯.png`
- **推荐尺寸**: 1920x400px
- **说明**: 显示在页面顶部的标头图片

### 2. 灯笼图片（汉字命名）
- **文件名格式**: `[灯名].png` （例如：平安灯.png、财运灯.png）
- **位置**: `/public/images/` 目录
- **推荐尺寸**: 800x800px 或正方形比例
- **格式**: PNG格式，支持透明背景

## 自动化功能

### 图片自动检测
- 系统会扫描 `/public/images/` 目录
- 自动识别包含中文字符的 `.png` 文件
- 从文件名提取灯的名称（去掉扩展名）
- 自动生成对应的灯笼卡片

### 动态配置
- 每个灯笼自动包含：
  - 灯名标签（显示在图片上方）
  - 灯笼图片
  - 价格（$19.9）
  - 描述文字（根据灯名自动生成）
  - 点灯按钮

## 页面布局

### 标头区域
- 全宽标头图片（祈福点灯.png）
- 渐变叠加层增强文字可读性
- 响应式高度（300-400px）

### 灯笼卡片
```
┌──────────────────────┐
│   灯名（金色标签）    │  ← 新增：图片上方的标签
├──────────────────────┤
│                      │
│   灯笼图片           │
│                      │
├──────────────────────┤
│  价格：$19.9        │

```

- **卡片容器**: 深紫渐变背景 + 金色边框 + 发光效果
- **神秘主题**: 与整体网站风格一致
- **响应式**: 移动端自适应布局
